## 2. External embeds and consent

The website may include embedded third-party content (for example Facebook, SlideShare, Scribd, or Google).

- Third-party embeds are not loaded automatically.
- Users first see a consent prompt.
- The embed is loaded only after the user chooses:
- `Load content` (only that embed), or
- `Allow all external embeds` (all embeds on the page).

If the user allows all external embeds, the choice is stored in browser `localStorage` with the key `external_media_consent`.

Please note: once third-party content is loaded, that service may set its own cookies or collect data under its own privacy policy.

## 3. Local storage (`localStorage`)

The site uses browser local storage for functional settings:

- `theme`: selected color theme (dark/light).
- `a11y-settings`: accessibility settings (for example text size, high contrast, reduced motion).
- `external_media_consent`: whether the user has allowed external embeds.

These values are not used for advertising or profiling.

## 4. Contact form

The contact form uses Formspree to deliver messages.
Submitted data is used only for handling contact requests.

## 5. Data retention

Browser local settings remain on the user's device until the user deletes or changes them.

Contact form data is retained only as long as needed to process the request.

## 6. Your rights

You can request more information about processing or request deletion of your data by contacting the controller.

## 7. Data Protection Officer (DPO) and request channel

No separate Data Protection Officer has been appointed for this website.

Data subject rights requests (for example access, rectification, or erasure requests) can be sent via:

- email: <jari.laru@oulu.fi>
- contact form: [www.jarilaru.fi/en/contact](https://www.jarilaru.fi/en/contact/)
